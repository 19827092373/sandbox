# Sandbox - 随手项目集

这个仓库用于存放平时开发过程中的各种独立项目、小样和练习。

## 目录结构规划

- `frontend-demos/`: 存放前端相关的项目、UI 练习和 Demo。
- `backend-lab/`: 存放后端 API、爬虫脚本、工具函数等。
- `scratchpad/`: 存放临时的代码片段或实验。
